# PRD SUMMARY: 연금부자서비스 (연금 투자 코치 v2)

## 1. 제품명
- 노후를 편안하게 연금부자서비스 (부제: 연금 투자 코치 v2)

## 2. 한 줄 요약
- 사용자의 연금 자산 배분을 AI가 진단하고, 위험성향 및 은퇴 시점에 최적화된 장기 모델 포트폴리오와 섹터 비중을 제안하는 서비스.

## 3. 해결하는 문제
- **정보 과잉과 선택 장애**: 수많은 투자 상품 중 본인의 연금 계좌에 적합한 상품과 비중을 결정하기 어려움.
- **포트폴리오 관리 부재**: 본인의 현재 자산 배분이 위험성향에 맞는지, 장기적으로 적절한지 판단할 기준이 부족함.
- **섹터 투자 난이도**: 유망 섹터를 알고 있어도 연금 계좌 내에서 어느 정도 비중으로 담아야 안전한지 알기 어려움.

## 4. 타깃 사용자
- **3040 직장인**: 연금 저축 및 IRP를 처음 시작하거나 효율적으로 운영하고 싶은 사용자.
- **4050 DC 전환자**: 확정기여형(DC) 퇴직연금으로 전환 후 직접 운용에 어려움을 느끼는 사용자.
- **은퇴 임박자(55세+)**: 안정적인 수령과 인컴 창출을 위한 포트폴리오 재편이 필요한 사용자.

## 5. 핵심 차별화 포인트
- **섹터 비중 번역기**: 단순히 유망 섹터를 나열하는 것이 아니라, 사용자의 전체 연금 포트폴리오 내에서 '최대 몇 %'가 적정한지 가이드 제공.
- **휴리스틱 + AI 하이브리드**: 프론트엔드의 안정적인 금융 규칙(휴리스틱)과 Gemini 3 Flash의 유연한 코칭을 결합하여 신뢰도 높은 결과 제공.

## 6. MVP 핵심 기능
- **진단 대시보드**: 현재 자산 배분 현황과 목표 포트폴리오를 한눈에 비교.
- **개인화 코칭 엔진**: 은퇴까지 남은 기간과 투자 성향을 고려한 맞춤형 리밸런싱 가이드.
- **상담 이력 시스템**: 과거 진단 내역을 추적하고 데이터로 관리(CSV 추출).

## 7. Out-of-scope (이번 MVP 제외 항목)
- 마이데이터 자동 연동 (수기 입력 방식 채택)
- 개별 종목(주식/ETF) 추천 및 자동 주문
- 초정밀 세금 시뮬레이션
- 실시간 시장 데이터 연동

## 8. 기술 스택 및 배포
- **Frontend**: React, TS, Vite, Tailwind CSS v4, Recharts
- **Backend**: Supabase (DB & Storage)
- **AI**: Gemini 3 Flash (`gemini-3-flash-preview`)
- **Deployment**: Vercel
